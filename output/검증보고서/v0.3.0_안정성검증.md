### B (round-trip 보존)

각 페이지에서 샘플 HTML 로드 → 편집 없이 "저장" → 페이지 새로고침 → "불러오기" → 편집기·`onChangeHtml 출력` 영역의 DOM 구조·텍스트·속성·중첩 일치 확인.

| 페이지 | 확장 | 특이 확인 |
|--------|------|---------|
| `/text` | headings · lists · links | 헤딩 레벨, 중첩 리스트, 링크 href·rel |
| `/code` | codeBlock · taskList | 코드블록 언어 클래스, 태스크 체크 상태 |
| `/images` | images | 더미 업로드 → src·alt·figure 구조 |
| `/tables` | tables | **colspan/rowspan 보존 필수** |

### C (기본 명령 동작)

| 페이지 | 명령 |
|--------|------|
| `/text` | H1↔H2↔H3 전환, 중첩 리스트 들이기/내보내기, 링크 단축키/다이얼로그 |
| `/code` | 언어 지정·줄바꿈, 태스크 체크박스 토글·중첩 |
| `/images` | 더미 훅으로 업로드·삽입·alt 입력 |
| `/tables` | "표 명령 toolbar" 버튼으로 — 표 삽입 → 셀 안에 커서 → 행/열 추가·삭제, 셀 병합(두 셀 드래그 선택 후), 셀 분리. `Tab`/`Shift+Tab`은 셀 이동 |

### F (부가 props)

`/basics` 페이지 상단 "F 영역 — 부가 props 검증 토글" 섹션:

| props | 확인 방법 |
|-------|----------|
| `placeholder` | input 입력값 변경 → 빈 에디터에 안내 문구 갱신 |
| `readOnly` | 체크 → 입력 차단·도구 비활성 |
| `autofocus` | 체크 후 `placeholder` 변경 → 재마운트되며 마운트 즉시 포커스 |
| `className` | DevTools Elements에서 `div.rte-editor.verify-classname-applied` 확인 |
| `onEditorReady` | 페이지 진입/재마운트 시 콘솔에 `[F-onEditorReady]` 로그 1회 |
