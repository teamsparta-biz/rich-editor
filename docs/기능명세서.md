# rich-editor — 기능명세서

> 최종 갱신: v0.5.0 기준 (확장 팩 8종 + 인라인 마크 6종)

## 1. 시스템 개요

### 정체성
- **라이브러리**: React 리치텍스트 에디터 (`@teamsparta-biz/rich-editor`)
- **소비처**: 팀스파르타 내부 서비스 (교안·콘텐츠 저작, 운영 도구 등)

### 기술 스택
- TipTap v2 (ProseMirror) · TypeScript strict · React 19
- 빌드: Vite library mode + tsup (ESM + CJS + .d.ts)
- 모노레포: pnpm workspace (`packages/rich-editor` + `apps/playground`)

### 배포 구조
- 패키지: npmjs (public) — `https://registry.npmjs.org`
- playground: Vercel

### 모듈 구조
```
packages/rich-editor/src/
├── index.ts              # public API 진입점
├── Editor.tsx            # React 래퍼 컴포넌트
├── extensions/           # 기본 TipTap 확장 팩
├── serialization/        # HTML ↔ 에디터 상태 변환
└── types.ts              # 공개 타입
```

## 2. 인증 및 권한

> 라이브러리 특성상 자체 인증 없음. 소비자 앱이 인증 책임.

## 3. 데이터 모델

> 라이브러리에는 DB 엔티티가 없음.
> 대신 **저장 포맷**을 데이터 모델로 취급: HTML 중심 (ProseMirror JSON은 필요 시 보조).

---

## 4. API 엔드포인트 명세

> 라이브러리에는 HTTP 엔드포인트 없음.
> 대신 **공개 컴포넌트 API**를 이 섹션에 기술한다.

### 공개 컴포넌트 (v0.5.0 기준 9확장)

```tsx
import { RichEditor } from '@teamsparta-biz/rich-editor'
import '@teamsparta-biz/rich-editor/styles.css'

<RichEditor
  initialHtml={value}
  onChangeHtml={setValue}
  extensions={['headings', 'lists', 'links', 'codeBlock', 'taskList', 'images', 'tables', 'marks']}
  readOnly={false}
  placeholder="내용을 입력하세요"
/>
```

### 확장 키 카탈로그 (v0.5.0, 9종)

| 키 | Phase | 주요 옵션 (요약) |
|----|-------|-----------------|
| `core` (기본 포함) | 11 | `placeholder?` |
| `headings` | 12 B1 | `levels?` (default `[1,2,3]`) |
| `lists` | 12 B1 | `nestedLists?` (default `true`) |
| `links` | 12 B1 | `openOnClick? / autolink? / defaultProtocol? / HTMLAttributes?` |
| `codeBlock` | 12 B1 | `languages? / defaultLanguage?` (lowlight common) |
| `taskList` | 12 B1 | `nestedTasks?` (default `true`) |
| `images` | 12 B2 | `imageUpload` 의무 / `allowCaption? / allowPaste? / allowDrop? / acceptedTypes? / maxSize?` |
| `tables` | 12 B2 | `allowMerge? / defaultRows? / defaultCols?` |
| `marks` | **14** | `bold? / italic? / strike? / code? / underline? / highlight?` (전부 default `true`) · `highlightColor?` (default `'#fef08a'` 노랑) — Bold/Italic/Strike/inline Code/Underline/Highlight 6종 단축키·마크다운 입력 단축어 자동 |

> 옵션 타입 9종은 패키지에서 `MarksExtensionOptions` 등 인터페이스로 export됩니다. 자세한 시그니처·CSS 변수(`--rte-highlight-bg`) 등은 [packages/rich-editor/README.md](../packages/rich-editor/README.md) 참조.
